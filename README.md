# Mavuno Water Solutions — website

This is a **static** (no build tools) multi-page website:

- `index.html` — Home
- `about.html` — About Us
- `solutions.html` — Our Solutions
- `impact.html` — Impact
- `get-involved.html` — Get Involved
- `contact.html` — Contact

## Preview locally

- **Fastest**: double-click `index.html` to open in your browser.
- **Recommended (more realistic)**: run any static file server from this folder (optional).
